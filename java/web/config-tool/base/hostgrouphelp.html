Host Group Help

Host Groups allow you to logically group together one or more Host objects. Generally this information is used for display on the Blue Console, which gives you an overview of your Host monitoring situation. Host Groups can be useful when you have numerous Hosts that perform synonymous tasks and can be thought of as a single entity i.e. All Database Servers.

To define a Host Group you must provide the following information:

  • Group Name: - The name of the Host Group
  • Alias: - The Alias of the Host Group
  • Members: - The Hosts that are to be members of this Host Group. This can be one or more Hosts.
