Service Extended Info Help

This page allows you to define Service Extended Info objects within your Blue monitoring configuration. Service Extended Info objects really provide no functionality other than to make the representation of your Services within the Blue Console look better. You can also attach further information to a Service through the use of action and notes URLs that can point to other websites. When defining Service Extended Info, you must provide the following information:

  • Host Name - The name of the Host on which this Service resides.
  • Service Description - The Description of the Service on the above Host.

You have the ability to specify several pieces of information to improve the view of your Service information. For instance you can specify notes to be used in conjunction with the Notes URL, and also give a new icon to the Service to make it more clearly identifiable within the Blue Console.

It should be noted that Service Extended Information is an optional part of your monitoring setup.
