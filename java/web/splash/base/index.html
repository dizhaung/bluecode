Welcome To Blue - A Java Port of Nagios 2.7

Blue is a Java port of the popular monitoring software Nagios. By utilising Java, Blue is able to give you the functionality of Nagios on both the Windows and Unix environments.

This small website is designed to give you information on how Blue runs and how to get the most out of your installation. For further in-depth information we suggest that you visit the Official Blue Website.
