Java Requirements:

As Blue is written in Java, you will at least require the Java Runtime Environment (JRE) for Blue to function properly. If you wish to develop your own plugins or extend the functionality that Blue offers, you will also require the Java Development Kit.

To install the JRE you will first need to download the latest version from the Website of Sun Microsystems. You can download the latest JRE here. Blue has been tested with JRE 1.6 and seems to work without issue. If you do not currently have Java installed then we recommend that you install the latest version of the 1.6 JRE.

With Java installed, you will need to add it to your path to faciliate the easy operation of Blue. There are numerous documents that explain the concept of a path and how to alter it to suit your needs. The following links explain how to alter your path under either the Windows or Unix environments:

  • Windows Path Information
  • Unix Path Information

If you currently have Java installed then Blue has been tested with Java 1.5 and above. It is currently not compiled for the 1.4 JRE and therefore you should consider upgrading to meet these requirements.
